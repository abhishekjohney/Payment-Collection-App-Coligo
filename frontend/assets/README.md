# Assets Folder

This folder contains app assets like icons and splash screens.

For a complete Expo app, you would need:
- icon.png (1024x1024)
- splash.png (1242x2436 or higher)
- adaptive-icon.png (1024x1024, Android)
- favicon.png (48x48, Web)

You can generate these using tools like:
- https://www.appicon.co/
- https://expo.github.io/styleguide/
- Or design custom icons in Figma/Photoshop

For now, Expo will use default placeholders.
